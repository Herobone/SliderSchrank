# SliderSchrank

This is an Android application that helps you choose your style of the day.

## Development Setup

### Environment

*   **IDE:** Android Studio (latest stable version recommended)
*   **Java:** JDK 17

### IDE Setup

1.  Clone this repository.
2.  Open the project in Android Studio.
3.  To ensure consistent code style, install the pre-commit hook by running the following command in the terminal:

    ```
    ./gradlew installGitHook
    ```

After these steps, you can build and run the app on an emulator or a physical device.
