# Contributing

source: https://www.conventionalcommits.org/en/v1.0.0/#summary

## Commits
* follow the scheme:  `((feat|fix|build|chore|ci|docs|style|refactor|perf|test)([scope, bspw. API/Camera/etc])): descriptive text`
* example: `feat(API): add API Connection timeout`

## Branches

* follow the scheme: `(feature|bugfix|hotfix|release|documentation)/scope_text`
* example: `feature/UI_add-button`
